# Customer

The Customer section allows you to manage all customer details, add new customers, and import bulk customer data into the system. Follow these steps to navigate and utilize the customer management features.

## Accessing Customer Management

>     Navigate to Sale > Customer.

<img src="../images/customer dashboard.png" alt="customer dashboard" style="border-radius: 10px; width: 70%; height: 70%; border: 0.5px solid #333;">

The Customer Dashboard will appear, displaying: A list of all customers, including business and individual types. A search bar to locate specific customers.

## Importing Bulk Customers

Click the Search button in the dashboard.<br>
Choose the Excel Operation option and download the Sample Import File by clicking the Click Here to Download Sample Import File link.<br>
Populate the downloaded file with customer data.<br>
Use the Import Excel option in the Customer Dashboard to upload the file and add customers in bulk.<br>

<img src="../images/sales return dashboard.png" alt="sales return dashboard" style="border-radius: 10px; width: 70%; height: 70%; border: 0.5px solid #333;">

## Adding a New Customer

Click Create New in the top-right bar to open the customer creation form. Fill out the following fields:

### Basic Details

Document Number Series: Automatically generated by the system. Configure via Document Number Series Settings.<br>
Customer Type: Select between:<br>
Business: Includes an additional field to select the Parent Customer from a dropdown.<br>
Individual: Standard fields are required.

### Adding an Address

Navigate to the Address Tab. Click Add Address to open the address form. Fill in the required address details and save.

### Saving the Customer
After completing all the fields, click Save. The new customer will now appear in the Customer Dashboard. The saved customer can be used for sales-related activities, such as generating delivery notes or invoices.
