# Sales Order

The Sales Order module enables businesses to create, manage, and process customer orders seamlessly. It integrates with the quotation and delivery workflows for streamlined operations.

## Accessing Sales Order Dashboard

>     Navigate to Sale > Sales Order.

<img src="../images/sale order dashboard.png" alt="sale order dashboard" style="border-radius: 10px; width: 70%; height: 70%; border: 0.5px solid #333;">

The Sales Order Dashboard will appear, displaying: All sales orders categorized as Draft or Approved.<br>
Options to Search or use Advanced Filters for refined results.

## Export to Excel

You can export all sales orders by clicking the Export to Excel button.

## Creating a New Sales Order

Click Create New in the top-right bar to open the Sales Order form. The process includes:

### Document Information

Document Number Series: Auto-generated by the system. Configure via the ðŸ‘‰ [Document Series Settings](<direct purchase.md>).<br>
Date: Specify the order date.

### Price and Customer Details

Price Type: Choose from Wholesale, Retail, or Regular. Configure via ðŸ‘‰ [Price Type Settings](<direct purchase.md>).<br>
Customer: Select the customer from the dropdown. Add customers via ðŸ‘‰ [Customer Menu](<direct purchase.md>).<br>
Displays any Due Amount for the selected customer.<br>
If the customer has any prior quotations, a Quotation Button will appear. Click it to view and select existing quotations.<br>

<div style="display: flex; gap: 10px;">
  <img src="../images/customer in sale order.png" style="width: 100%; border-radius: 10px; border: 0.5px solid #333;">

Upon selection, the system automatically populates the Item Details and prices from the chosen quotation.<br>
If the customer has an associated tax (e.g., UAE VAT, India GST), a Tax column will appear in the item section, and the total amount will adjust based on the subtotal.

### Additional Information

Store: Select the store for the sales order. Configure via ðŸ‘‰ [Store Settings](<direct purchase.md>).<br>
Salesperson: Assign a salesperson. Create salespersons via ðŸ‘‰ [Salesperson Settings](<direct purchase.md>).<br>
Expected Delivery Date: Specify the date when the order is expected to be delivered.
Project: Link the sales order to a specific project. Configure via ðŸ‘‰ [Project Settings](<direct purchase.md>).<br>

### Item Selection

If a quotation is selected, the item details will populate automatically.

## Saving and Approving Sales Order

Save: Once all details are entered, click Save to create the sales order.<br>
Approve: After saving, click Approve to finalize the order.

<img src="../images/approved so.png" alt="approved so" style="border-radius: 10px; width: 70%; height: 70%; border: 0.5px solid #333;">
