# Sales Quotation

The Sales Quotation module is designed to create, manage, and approve quotations for customers. It also integrates seamlessly with the sales order workflow.

## Accessing Sales Quotation Dashboard

>     Navigate to Sale > Sales Quotation.

<img src="../images/Sales Quotation dasboard.png" alt="Sales Quotation dasboard" style="border-radius: 10px; width: 70%; height: 70%; border: 0.5px solid #333;">

The Sales Quotation Dashboard will appear, displaying: All sales quotations categorized as Draft or Approved.<br>
Options to Search or use Advanced Filters for refined results.

## Export to Excel

You can export all sales quotations by clicking the Export to Excel button.

## Creating a New Sales Quotation

<img src="../images/Sales Quotation create new.png" alt="Sales Quotation create new" style="border-radius: 10px; width: 70%; height: 70%; border: 0.5px solid #333;">

Click Create New in the top-right bar to open the Sales Quotation form. The process includes:

### Document Information

Document Number Series: Auto-generated by the system. Configure via the ðŸ‘‰ [Document Series Settings](<direct purchase.md>).<br>
Date: Specify the quotation date.

### Price and Customer Details

Price Type: Choose from Wholesale, Retail, or Regular. Configure via ðŸ‘‰ [Price Type Settings](<direct purchase.md>).<br>
Customer: Select the customer from the dropdown. Add customers via ðŸ‘‰ [Customer Menu](<direct purchase.md>).<br>
Displays any Due Amount for the selected customer.<br>
If the customer has associated tax (e.g., UAE VAT, India GST), a Tax column will appear in the item section. The total amount is calculated automatically based on the subtotal.

### Salesperson and Project
Salesperson: Assign a salesperson. Create salespersons via ðŸ‘‰ [Salesperson Settings](<direct purchase.md>).<br>
Project: Link the quotation to a specific project. Configure via ðŸ‘‰ [Project Settings](<direct purchase.md>).<br>

### Item Selection
Item: Select the item from the dropdown list. The list displays Current Stock, Reorder Quantity, and Average Rate for the item. If the item doesnâ€™t exist, create it via ðŸ‘‰ [Item Menu](<direct purchase.md>).<br>
Enter Quantity, and the unit and rate will auto-fill based on the item's configuration. You can manually adjust the rate if required. Add more items by clicking Add Line.

### Financial Details
Subtotal: Automatically calculated.<br>
Discount: Apply a discount if applicable.<br>
Adjust Amount: Add or subtract adjustments.<br>
Add additional details such as Reference Number, Statement Reference, and Cost Center.<br>

## Saving and Approving Sales Quotation
Save: Once all details are entered, click Save to create the quotation<br>
Approve: After saving, click Approve to finalize the quotation.<br>
